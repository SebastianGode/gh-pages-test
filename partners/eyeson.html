eyeson API – Cloud-based Video Meetings

Combine live video, audio, and data in one interface with the cloud-based API from eyeson – easily integrable into your environment for greater efficiency and productivity.

Product type: PaaS

Successful customer engagement in any business workflow

eyeson is a European cloud video meeting service that improves remote team communication in day-to-day business, significantly simplifying video meetings. eyeson helps companies and segments to conduct highly secure video conferences and video events. The patented eyeson single-stream technology enables meetings to take place even in remote regions or where only limited bandwidth is available.

Patented Single Stream Technology

The eyeson single stream technology merges all live media, data and participants into a single video and audio stream in real time. This single connection to the web client means eyeson uses less bandwidth and CPU load than competitors.

Easy use & integration

No external apps or downloads - one click starts the meeting, no matter which browser or device. The predefined, open-source frontend also allows an easy and individually customizable integration.

Guaranteed security

The eyeson API is completely GDPR-compliant. Using the security-by-design approach, a temporary server is started in the cloud for each meeting, on which all data is automatically deleted after termination.
